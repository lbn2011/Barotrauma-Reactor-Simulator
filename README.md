# Barotrauma 反应堆模拟器

一个用于模拟和分析 Barotrauma 中核反应堆系统的工具。

## 概述

该模拟器为 Barotrauma 游戏提供了详细的核反应堆机械模型。它允许玩家在在游戏中实施之前设计、测试和优化反应堆配置。

## 功能特性

- 实时反应堆模拟
- 燃料消耗跟踪
- 热量产生和冷却计算
- 功率输出监控
- 安全系统建模
- 组件故障模拟

## 安装

1. 克隆仓库
2. 使用 `npm install` 安装依赖项
3. 使用 `npm run dev` 启动开发服务器

## 使用方法

1. 配置反应堆布局
2. 设置燃料棒位置
3. 调整冷却系统
4. 监控反应堆性能
5. 优化效率和安全性
